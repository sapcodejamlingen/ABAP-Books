  PUBLIC SECTION.
    CLASS-METHODS main
    IMPORTING  i_carrid TYPE spfli-carrid

*----------------------------------------------------------------------*

CLASS demo IMPLEMENTATION.
  METHOD main.
    DATA: spfli_tab TYPE TABLE OF spfli,
          msg TYPE string.

    SELECT * FROM spfli
      INTO TABLE spfli_tab
      WHERE carrid = i_carrid.
    msg = sy-dbcnt.
    CONCATENATE `sy-subrc: ` msg INTO msg.
    MESSAGE msg TYPE 'I'.
  ENDMETHOD.                    "main
ENDCLASS.                    "demo IMPLEMENTATION
